# TypeScript para Principiantes

TypeScript es un superconjunto de JavaScript que añade tipado estático. Ayuda a detectar errores temprano y mejora la mantenibilidad del código.

## Beneficios

- **Seguridad de Tipos**: Detecta errores relacionados con tipos en tiempo de compilación.
- **IntelliSense**: Mejor autocompletado y navegación en tu IDE.
- **Características Modernas**: Usa las últimas características de ECMAScript.
