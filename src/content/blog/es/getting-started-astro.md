# Empezando con Astro

Astro es un nuevo tipo de constructor de sitios estáticos para la web moderna. Ofrece un rendimiento ultrarrápido con una experiencia de desarrollador moderna.

## ¿Por qué Astro?

1. **Cero JS por defecto**: Astro renderiza HTML en el servidor y elimina cualquier JavaScript restante.
2. **Colecciones**: Gestiona tu contenido con seguridad de tipos.
3. **Integraciones**: Usa tus herramientas favoritas como Tailwind, React, Vue, etc.
