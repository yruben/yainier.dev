# Dominando Tailwind CSS

Tailwind CSS es un framework CSS de utilidad primero que te permite construir diseños personalizados sin salir de tu HTML.

## Conceptos Clave

- **Clases de Utilidad**: Clases de propósito único como `flex`, `pt-4`, `text-center`.
- **Diseño Responsivo**: Prefija utilidades con puntos de interrupción como `md:flex`.
- **Personalización**: Configura tu tema en `tailwind.config.mjs`.
