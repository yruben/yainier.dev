# Portafolio Personal

Este es el sitio web del portafolio que estás explorando ahora mismo. Está diseñado para ser una muestra rápida, accesible y visualmente impactante de mis habilidades y proyectos.

**Características principales:**
*   **Alto Rendimiento:** Construido con [Astro](https://astro.build/) para una arquitectura de cero JavaScript por defecto, asegurando tiempos de carga ultrarrápidos.
*   **UI/UX Moderna:** Estilizado con [Tailwind CSS](https://tailwindcss.com/) utilizando una estética de tema oscuro inspirada en neón (vibra "Cyberpunk/Futurista").
*   **Componentes Interactivos:** Aprovechando **React** para elementos de UI dinámicos como el efecto de máquina de escribir en el héroe, tarjetas de proyectos y animaciones (impulsadas por Framer Motion).
*   **Internacionalización (i18n):** Soporte completo para contenido en inglés y español.
*   **Modo Oscuro:** Cambio de tema integrado según la preferencia del usuario.
*   **Diseño Responsivo:** Diseño totalmente fluido que se ve genial en móviles, tabletas y escritorio.
